Widget by @AverageCaret 
Created 01/23/2023

streamelements widget that responds to
live events

Responds to followers, subscriber, gift subs, donations, twitch gift bits

Response parameters can be turned on or off and minimum amounts can be set 

custom characters for the scrambler can be used (see fields tab)

How to use:

first create a custom widget in the streamelements overlay editor

each file has a sufix, copy and paste the contents of each file 
into the tabs with the same names in the streamelements code editor

adjust font size, color and response parameters in the fields section
